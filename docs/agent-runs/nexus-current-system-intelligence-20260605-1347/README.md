# NEXUS Current System Intelligence

This folder contains the current-system intelligence run for NEXUS. It is static-first, source-cited, and privacy-safe.

- HTML report: `index.html`
- Markdown report: `report.md`
- Machine manifest: `machine-manifest.json`
- Completion report: `completion-report.md`
- Maps: `maps/`
- Context packs: `context-packs/`
- Machine data: `assets/data/` and `reports/**.json`

No production Supabase access was performed. No `src` business logic was modified.
