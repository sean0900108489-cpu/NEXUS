## Top-Level Architecture Summary

NEXUS // AI OPS is a **Next.js App Router** application using **Supabase** for persistence and **Zustand** for client-side state management. The architecture follows a **local-first** model with **cloud durability** via Supabase sync operations.

### Layering (L1-L4)

- **L1 — Agent Capabilities**: chat, image, video, sandbox, audio, search, data-analysis
- **L2 — Agent Handoff & Routing**: handoff rules registry (currently empty, planned)
- **L3 — Workspace Graph & Workflow Runtime**: React Flow graph, Workflow Runtime Lite, ops surface
- **L4 — System Operations**: observability, deployment checks, sync, admin, feature flags
