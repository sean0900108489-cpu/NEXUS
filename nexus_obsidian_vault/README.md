# NEXUS Workspace Floating App Runtime

這份 Obsidian vault 是重新校正後的 NEXUS 規劃版本。

舊方向把 `/desktop` / Window OS 當成主線，這不符合真正產品意圖。新的方向是：

**NEXUS 是 Workspace-first Floating App Platform。**

也就是：使用者登入後回到 NEXUS Workspace，所有未來新功能都以 workspace 內的 floating app 視窗接入，例如 Sandbox、Chat、Notes、Forum、Feed、Profile、Airtasker-like、Reddit-like、Canva-like、Marketplace。

## 核心結論

- Workspace 是主舞台。
- Floating App Runtime 是核心技術層。
- 所有新功能都是 NEXUS floating app。
- `/desktop` 只是 proof-of-concept / 實驗場，不是最終產品主入口。
- `/desktop` 做出的好東西要吸收回 Workspace Floating Runtime。
- Workspace 可以承載所有浮動 app，但 Workspace 本體不能變成 god file。
