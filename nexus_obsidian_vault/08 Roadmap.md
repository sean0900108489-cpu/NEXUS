# 08 Roadmap

## 新 Roadmap

### R0 Restore Login Landing

登入後回到原本 NEXUS / Workspace-first 入口，不自動進 `/desktop`。

### R1 Workspace Floating Runtime Inventory

盤點現有 workspace floating windows，包括 sandbox window。

### R2 Shared Floating Runtime Boundary Plan

設計 shared floating runtime，把 workspace 現有視窗能力與 `/desktop` proof-of-concept 能力合併。

### R3 Registry Bridge Into Workspace

讓 Workspace 可以透過 registry 開啟 floating apps，而不是硬寫 app rendering。

### R4 Migrate POC Apps Into Workspace Floating Apps

把 Notes、Artifact、Profile、Feed、Developer Inspector 等 POC app 接回 Workspace floating runtime。

### R5 First Real Workspace Floating Product Prototype

選一個產品原型，例如 Reddit-like 或 Airtasker-like，作為 floating app 測試。

### R6 Runtime Hardening

處理 permissions、resource routing、layout persistence、performance、mobile、keyboard shortcuts。

## 暫緩事項

- 完整 Marketplace MVP
- 完整 Reddit clone
- 完整 Canva editor
- DB migration
- public social graph
- payments

先穩定 runtime，再做大型產品。
