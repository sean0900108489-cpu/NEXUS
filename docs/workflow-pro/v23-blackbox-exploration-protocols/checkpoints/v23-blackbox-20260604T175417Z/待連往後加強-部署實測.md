## Why Deployed Probes Matter

- `NODE_ENV` and `VERCEL_ENV` affect route behavior, legacy-tool blocking, deployment classification, provider authority, and workspace session selection.

- Production image generation should prove app auth and workspace permission before spending provider calls.
- UI claims about Workflow Pro, generated history, download, import/export, and account actions require Computer Use screen evidence on the actual target URL.
