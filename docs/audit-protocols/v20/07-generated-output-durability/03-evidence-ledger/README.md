# Evidence Ledger Workspace

Place redacted evidence summaries here during the post-fix scan.

Acceptable evidence:

- Supabase schema facts
- aggregate counts
- join counts
- boolean probe outcomes
- status-code matrices
- test command names and pass/fail summaries
- Vercel/GitHub deployment ids and status summaries

Do not store:

- raw message content
- raw artifact content
- cookies or Authorization headers
- Supabase keys or provider keys
- full browser IndexedDB/localStorage dumps
- full private response bodies
