# 10 Visual Design Brief

## 目標

index.html 要像高級產品策略報告 + 系統控制台 + 架構圖譜，不像普通 README。

## 內容區塊

- Hero / Cover
- Executive Summary
- Project Health Snapshot
- Architecture Map
- Supabase Connection Map
- Nexus Domain Map
- Large File Risk Zone
- State Store Contract Map
- Dependency Graph
- Dead Code / Unknowns
- Refactor Playbook
- Agent Context Packs
- Round Logs
- Completion Report

## 互動需求

- Anchor nav。
- Human / Agent summary toggle。
- Risk severity filter。
- Large file cards。
- Local image assets，無 CDN。
- Machine manifest link。

## 安全

HTML 不包含 secrets，不連外，不上傳任何資料。
