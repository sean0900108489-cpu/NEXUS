# Evidence Ledger Workspace

Use this folder for redacted evidence gathered during the post-fix scan.

Allowed evidence:

- command names and status codes
- counts
- booleans
- file paths and line references
- schema and policy summaries
- redacted error codes

Disallowed evidence:

- raw tokens
- provider keys
- service-role keys
- cookies
- localStorage or IndexedDB dumps
- full database rows
- unredacted Authorization headers
